审查报告

双审融合审查报告 v2.0

LUMI-ARCH-002。22 项发现 (6 关键 / 12 中等 / 2 低 / 2 信息)，16 份文档清单，审查约束响应。

微架构审查报告 v1.0

LUMI-ARCH-002 初版。流水线 / FU / 旁路 / 冒险 / GEM5 交叉校验。

Batch-1: 流水线核心模块 5 docs

Fetch & Branch Predictor

F1/F2 级取指 + LTAGE 分支预测 + BTB/RAS。双发射取指，分支误预测惩罚 ≤3 cycle。

LUMI-DESIGN-001

Decode & Issue

D/I 级三发射解码 + 冒险检测 + skip-stalled 优化。支持 RV32IMACFV 全指令集。

LUMI-DESIGN-002

Execute (ALU/MUL/DIV)

E1/E2 级 ALU + MUL/DIV 执行单元。旁路网络 + 多周期除法状态机。

LUMI-DESIGN-003

Writeback & CSR/HPM

W 级写回 + CSR 寄存器堆 + 29 个 HPM 事件计数器。D-013 中断可见性约束。

LUMI-DESIGN-004

Register File

整数 (x0~x31) / 浮点 (f0~f31) / 向量 (v0~v31, VLEN=128) 寄存器堆。D-006c 物理端口约束。

LUMI-DESIGN-005

Batch-2: 存储与总线接口 3 docs

TCM (Tightly Coupled Memory)

ITCM/DTCM 紧耦合内存，ECC 保护 + DMA 引擎。gem5 延迟 ITCM=2/DTCM=3。

LUMI-DESIGN-006

L1 Cache (I-Cache / D-Cache)

L1 I/D-Cache 各 32KB（可配 8/16/32KB），4 路组相联，ECC + RVWMO 一致性。

LUMI-DESIGN-007

PMA & AXI Bus Interface

PMA 检查器 + AXI4 总线接口 + FFP 快速路径。8 区域 PMA 配置。

LUMI-DESIGN-008

Batch-3: 中断 / 异常 / 特权 / 调试 4 docs

CLIC + PLIC 中断控制器

两层中断仲裁，CLIC 16 cycle 硬件延时，PLIC 兼容模式。64 级优先级。

LUMI-DESIGN-009

异常处理与陷阱委托

精确异常 + M/S 级委托 + NMI 不可屏蔽中断。LUMI-SYS-002/003 延时约束。

LUMI-DESIGN-010

特权架构 & PMP

M/S/U 三级特权 + PMP 物理内存保护 + Smepmp ePMP 增强。16 区域 PMP 条目。

LUMI-DESIGN-011

Debug & Trace

JTAG Debug Module + Nexus Trace + Trigger Module。DMI 接口 + 硬件断点。

LUMI-DESIGN-012

Batch-4: 扩展功能模块 4 docs

FPU & Vector 单元

F/D 双精度浮点 + RVV 1.0 向量扩展。D-006a/b/c FU 约束 + DEC-007-2 三接口。

LUMI-DESIGN-013

低功耗 & 复位 & 时钟域

WFI/WRS 低功耗 + ICG 时钟门控 + 多域复位 + UPF Power Intent。LUMI-PWR-001/002。

LUMI-DESIGN-014

多核互联 & 双核锁步

1~4 核互联 + 双核锁步比较器。D-011 双级比较约束 + Cache Coherence 占位。

LUMI-DESIGN-015

DFT & 安全 & 物理约束

Scan Chain + MBIST + JTAG + SDC 时序约束。LUMI-AREA-001 面积约束。

LUMI-DESIGN-016

LLD 详细设计文档 LLD

LLD 文档索引

详细低层设计文档集合，覆盖各子模块的 RTL 实现细节、接口约束、时序分析和验证策略。与 HLD 文档形成双层设计体系。

LLD-INDEX

模块间接口契约

全模块交互接口精确定义：流水线级间、存储、总线、中断、调试、扩展接口 + 全局信号传播矩阵。v2.1 scalar 端口版本，附录 C 列出 26 项未连接/预留接口。

LUMI-LLD-IC

HLD ↔ LLD 双向追溯映射表

16 份 HLD 文档映射到 20 份 LLD 文档。HLD 定义架构规格，LLD 定义 RTL 信号级实现。4 份 LLD 为新增（无直接 HLD 对应）。
