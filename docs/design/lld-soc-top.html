4. 状态机定义

本模块无显式 FSM。状态机分散在各子模块中: u_axi (AXI 协议 FSM), u_csr (CSR 访问), u_power (电源状态机), u_dft (scan 控制)。SoC 顶层仅做连线和集成。

6.3 未定义行为声明

  • V1 SRAM 索引超出 [0:65535] 范围: 地址截断, 无异常生成
  • FPU/Vector issue_valid 固定为 0: 协处理器不参与执行, 结果未定义
  • PLIC ext_irq_in 固定为 32'h0: 外部中断不通过 PLIC, 仅 CLIC 有效
